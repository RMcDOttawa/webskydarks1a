Web SkyDarks Help

This program is a controller to help take dark frames for astrophotography using TheSkyX Professional from Software Bisque. It is useful only if you have TheSkyX Pro, and are using it to control your camera.

Note that the ability to connect to TheSkyX in this manner is a feature of the TheSkyX Professional version. TheSkyX Serious Observer does not have this feature and cannot be used.

As you can see, this program is running in your web browser. It runs only in your browser, and it stores intermediate data in your browser. There is no “back end” server, so your data are not exposed to the outside world. This means that if you wish to repeat or a continue a session, you will need to do so on the same browser on the same computer.
