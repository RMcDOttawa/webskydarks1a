The Frames Plan page

The Frames Plan is where you define and manage the list of all the dark and bias frames that will be collected once you start the collection process from the Run Session page. Here is an example:

Each row in this table is a Frame Set, which is a set of some number of identical frames. For example, a frame set might be "16 bias frames binned 1x1", or "24 dark frames, 60 seconds each, binned 2x2".

If the frames in a frame set are identical, why would you need more than one? That explanation is beyond the scope of this help file, but the quality of your image calibration is improved by having multiple dark or bias frames captured with the same camera settings, and your image processing software will have the ability to accept the multiple frames and average them. Generally, the more frames in a frame set the better, but there are "diminishing returns" that must be balanced against the storage required for the frames. I usually use 16 to 32. (There is no reason the number has to be a power of 2; I just like powers of 2.)

Managing the list of frame sets

You can manipulate the rows in this table as you would expect.

  • Click on a row to select it. Shift-click to toggle selection.
  • Click the pencil icon, or just double-click a row, to edit it.
  • Click the "-" button to delete selected rows.
  • Click the up- and down-arrows to move rows in the table.
  • To add a row to the table, click the "+" button. If a row is already selected, the new one will be inserted above the selected row. If no row is selected, the new one will go at the end.

Quickly adding multiple frame sets to the list

Defining your list of frame sets one at a time would be tedious, and there is a fast way to generate a large list.

Click the "bulk add" button library_add to access the "Bulk Add" window:

In this window you can quickly specify a large number of frame sets.

To Generate Multiple Bias Frames

  • fill in the number you want in the top section, and
  • select one or more checkboxes to indicate what binning you want.

If you don't want any bias frames, be sure to set the number to zero.

To Generate Multiple Dark Frames

  • fill in the number you want in the top section,
  • select one or more checkboxes to indicate what binning you want, and
  • list the exposure times you want, separated by spaces, in the exposure lengths field.

If you don't want any dark frames, be sure to set the number to zero.

Example

The example shown above will generate 12 frame sets, requesting:

  • 16 Bias frames binned 1x1,
  • 16 Bias frames binned 2x2,
  • 16 Dark frames, 30 seconds exposure, binned 1x1,
  • 16 Dark frames, 30 seconds exposure, binned 2x2,
  • 16 Dark frames, 60 seconds exposure, binned 1x1,
  • 16 Dark frames, 60 seconds exposure, binned 2x2,
  • 16 Dark frames, 180 seconds exposure, binned 1x1,
  • 16 Dark frames, 180 seconds exposure, binned 2x2,
  • 16 Dark frames, 300 seconds exposure, binned 1x1,
  • 16 Dark frames, 300 seconds exposure, binned 2x2,
  • 16 Dark frames, 600 seconds exposure, binned 1x1, and
  • 16 Dark frames, 600 seconds exposure, binned 2x2.

When you click "Save" the frames will all be added to the frame plan table, where you can then manipulate it further with the edit buttons described above.
