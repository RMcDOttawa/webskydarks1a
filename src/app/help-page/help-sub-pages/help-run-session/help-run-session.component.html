The Run Session page

The Run Session tab is where you begin, monitor, and control the acquisition of your dark and bias frames.

There are 3 main sections to this page:

Images Being Acquired

This table shows a summary of your frames plan. This is informatino only - the frames plan cannot be modified from here; you must go back to the Frames Plan tab to do that. The table is annotated to show the progress of the acquisition.

  • Lines shaded in gray are complete (the frames have been acquired).
  • Lines not shaded have not been started yet.
  • The line shaded in green is the frame set for which acquisition is in progress or which will begin when the Begin button is clicked.

In the example above, the first frame set is complete, the second frame set is in progress, and the rest have not been started.

This table is kept up to date as acquisition proceeds, and is saved between sessions. So, if you interrupt acquisition, you can resume later and it will pick up from where it left off.

Console Log

A series of information messages will appear here as acquisition proceeds, giving you status updates on the progress.

Buttons

Begin Session This button is enabled only if there is a frames plan with images that still need to be acquired. When it is clicked, acquisition will start or, if a delayed start has been specified, waiting for the start time will begin.
Cancel Session This button is enabled only when an acquisition is running. It will immediately end the image acquisition, and whatever frame is in progress will be lost. Any work already done is not lost, and the session can be resumed later.
Clear Console This clears the lines from the console log, but has not effect on the acquisition process or frames plan.

Where Images are Saved

It's important to remember that images are saved by TheSkyX on the computer where TheSkyX is running. Not the computer where you are running WebSkyDarks (unless they happen to be the same computer). The TheSkyX does not offer the service of sending acquired images across the network - nor would you want it to, as they are usually very large.

If the server and relay are running, the path at which TheSkyX will be saving images is displayed at the top of the run-session page, above the console log. This is for information only. You can't change that location here; you must change it from within TheSkyX.
